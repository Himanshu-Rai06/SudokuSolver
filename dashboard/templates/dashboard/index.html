About

Abstract

This Sudoku engine is a high-performance, AI-driven solver built entirely from scratch. Designed as a capstone project by Himanshu Rai, the objective was to create an algorithm capable of solving thousands of puzzles per second using pure Python, with absolutely zero external libraries or dependencies.

Methodology & Techniques

Instead of relying on basic brute-force, this engine mirrors advanced human logic combined with machine efficiency. It operates on a three-tier architecture:

  • Constraint Propagation: The engine continuously scans rows, columns, and 3x3 grids to eliminate impossible values, instantly filling "Naked Singles" (cells with only one mathematical possibility).
  • Frequency Analysis (Hidden Singles): The algorithm tracks the most frequently occurring digits on the board to deduce placements that might not be obvious to a standard row/column scan.
  • Heuristic Backtracking (MRV): When pure logic hits a wall, the engine employs a Minimum Remaining Values (MRV) heuristic. It maps out the board, finds the cell with the fewest possible remaining options, and branches out from there. This drastically prunes the decision tree and prevents the algorithm from wasting time on dead ends.

Developer Tools & Benchmarking

To prove the efficiency of the algorithm, a Live Engine Benchmark is included in the Developer Tools menu. This allows users to pull subsets from a 1,000,000+ puzzle dataset directly into server RAM and test the Python engine's solve speed in real-time, often averaging less than 1 millisecond per puzzle.

How to Play & Features

The Basics

The goal is simple: fill the entire 9×9 grid so that every row, column, and 3×3 box contains the numbers 1 through 9 exactly once. No duplicates are allowed in any of these areas.

App Controls

  • Theme: Tap the top-left icon to switch up the visual style of the app.
  • Restart: Single tap to wipe your current progress and restart the puzzle. Double tap to generate a completely new puzzle.
  • Hint / Solve: Single tap reveals the correct number for your currently selected cell. Double tap commands the AI to solve the entire board for you.
  • Settings & Data: Tap the gear icon to access game settings. Here you can Save/Load your current board state, change difficulty loadouts, and toggle features.
  • Timer: Keep track of your solving speed. Compete for the best times on the Leaderboard!
